# 分类

### Message Digest

##### MD4

- 1990年设计
- 128位
- 不安全

##### MD5

- 1991年设计
- 128位
- 不安全（2004年攻破，王小川）

### Secure Hash Algorithm

##### SHA-1

- 1993年诞生
- 160位
- 不安全（2005年攻破，王小川）

##### SHA-2

- 2002年诞生
- SHA-256
- SHA-384
- SHA-512
- 安全

##### SHA-3

- 2007年开始公开征集
- 2012年诞生
- 进过5年选拔
- Keccack脱颖而出
- 其中Joan Deamen也是AES的设计者
